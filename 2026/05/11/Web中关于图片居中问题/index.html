Web中关于图片居中问题

一、文本居中法：仅实现图片水平居中
原理:img 标签属于行内块元素，和文字特性类似，给图片父容器设置文本居中，就能让图片水平居中。
代码示例
html代码

css .img-box { /* 核心：水平居中 */ text-align: center; } 优点：代码极简、兼容性极好、不用设置宽高 缺点：只能水平居中，无法实现垂直居中 二、margin 自动居中法：块级图片水平居中 原理:把图片转为块级元素，再设置左右外边距自动分配，实现水平居中。 代码示例 css img { display: block; /* 转为块级元素 */ margin: 0 auto; /* 左右自动居中 */ } 优点：不用依赖父元素、自适应宽度 缺点：依旧只能水平居中，不能垂直居中 三、Flex 弹性布局：推荐！水平 + 垂直一键居中 原理:给图片父容器开启 Flex 弹性布局，通过两条属性分别控制水平、垂直居中，是目前开发首选方案。 代码示例 html代码
css .flex-box { width: 100%; height: 400px; /* 父容器需要固定高度才看得出垂直居中 */ border: 1px solid #eee;

/* 核心居中三行代码 /
display: flex;
justify-content: center; / 水平居中 /
align-items: center; / 垂直居中 */
}
优点：写法简洁、语义清晰、适配所有现代浏览器、响应式友好
缺点：需要给父容器设置高度才能看到垂直效果
四、绝对定位 + 位移居中：脱离文档流居中
原理:父容器设置相对定位，图片设置绝对定位，先拉到容器 50% 位置，再通过 transform 往回偏移自身宽高的一半，实现正中心居中。
代码示例
html
预览

css .pos-box { position: relative; /* 父相 */ width: 100%; height: 400px; border: 1px solid #eee; } .pos-box img { position: absolute; /* 子绝 */ top: 50%; left: 50%; transform: translate(-50%, -50%); } 优点：不占文档流空间、精准居中 缺点：定位层级容易受影响，布局复杂时需注意层级覆盖
